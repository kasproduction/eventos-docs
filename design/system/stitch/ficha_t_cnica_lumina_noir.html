Componentes & Layout (Specs) ### Tarjetas (Cards) - **Fondo:** `#0E0E0E` o `rgba(255,255,255, 0.03)` - **Borde:** `1px solid rgba(255,255,255, 0.08)` - **Radio de Esquina (Radius):** `24px` (Premium Roundness) - **Sombra:** Ninguna (usar elevación tonal a través de bordes y contrastes) ### Botones Principales (CTA) - **Fondo:** `#D1FC00` - **Texto:** `#000000` (Bold) - **Radius:** `100px` (Full Round) - **Padding:** `16px 24px` ### Navegación (Bottom Nav) - **Fondo:** `#0E0E0E` con `backdrop-filter: blur(20px)` - **Icono Activo:** `#D1FC00` dentro de círculo `#D1FC00` (opacidad 10%) - **Icono Inactivo:** `rgba(255,255,255, 0.4)` --- ## 4. Iconografía & Estilo Visual - **Estilo:** `Material Symbols` o `Lucide Icons`. - **Grosor (Weight):** `Rounded` o `Fill` según el contexto de importancia. - **Efectos:** Uso de `Glow` (Box shadow con color neón) muy sutil (0 0 15px) solo para elementos "En Vivo" o indicadores críticos. - **Imágenes:** Filtro de desaturación leve o "Dark Overlay" para asegurar la legibilidad del texto blanco sobre ellas. --- *Ficha generada para el proyecto EventOS — Lumina Noir v1.0*
